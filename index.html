3D Touch开发

发表于 2016-06-02   |   分类于 iOS   |  

前言

3D Touch的触控技术，被苹果称为新一代多点触控技术。其实，就是此前在Apple Watch上采用的Force Touch，屏幕可感应不同的感压力度触控。首次使用在2015年9月发布的iPhone 6S上。
主要应用
第一部分的应用是我们可以通过3D手势，在主屏幕上的应用Icon处，直接进入应用的响应功能模块。这个功能就例如我们上面的日历示例，会在icon旁边出现一个菜单，点击菜单我们可以进入相应的功能单元。
第二部分是对app的一个优化，用户可以通过3D Touch手势在view上来预览一些预加载信息，这样的设计可以使app更加简洁大方，交互性也更强。

使用Hexo+Github搭建自己的博客

发表于 2016-06-01   |   分类于 hexo   |  

前言

看惯了国内在线博客千篇一律的样子，博主毅然决然的打算自己搭建一套静态博客。某日在掘金上发现了一篇基于hexo与github的静态博客搭建的文章，大爱这个简单又扩展方便的hexo。直到今日有时间便花了点时间搭建。

前期环境搭建

由于使用的是hexo与github page的搭配，所以首先需要安装git。

安装Git

通过Homebrew安装Git

$ brew install git

或者通过GitHub for Mac网站下载该工具。

安装Node.js

使用Homebrew安装node.js 

$ brew install node

检查node.js是否已经安装完成 

$ node --version

如果看到版本号，那么node.js就算安转完成了。

配置GitHub Page

首先注册一个github帐号 废话，然后新建一个repository，注意repository name必须为username.github.com
例如你的用户名是Evrins那么你的repository name就是Evrins.github.com新建完repository之后点击repository的Settings在GitHub Pages上将这个页面自动生成。如果在游览器输入http://evrins.github.io/能看到初始化的页面，那么前期的搭建工作就算成功了。

安装Hexo

由于前期安装了node.js，所以安装hexo的时候就简单很多。使用终端输入 

$ sudo npm install hexo-cli -g

这样hexo本体就算安装完成了。为了更好的扩展hexo的功能，也可以在这时侯安装一部分的插件 

$ npm install hexo-generator-index --save  
$ npm install hexo-generator-archive --save  
$ npm install hexo-generator-category --save   
$ npm install hexo-generator-tag --save
$ npm install hexo-server --save
$ npm install hexo-deployer-git --save
$ npm install hexo-deployer-heroku --save
$ npm install hexo-deployer-rsync --save
$ npm install hexo-deployer-openshift --save
$ npm install hexo-renderer-marked@0.2 --save
$ npm install hexo-renderer-stylus@0.2 --save
$ npm install hexo-generator-feed@1 --save
$ npm install hexo-generator-sitemap@1 --save

为了发布到github page，hexo-deployer-git是必须安装的插件。

配置Hexo

在终端输入
mkdir hexo
创建hexo的主文件夹
cd hexo
进入hexo文件夹
hexo init
初始化hexo
npm install
安装依赖包
hexo server
然后在游览器中输入localhost:4000,查看hexo在本地运行的情况。

配置全局 _config.yml

在hexo文件夹下，有_config.yml的文件，使用记事本等软件打开之后可以看到hexo的配置情况。具体配置情况可以再hexo的官网来进行查阅。主要说下关于发布到github上的几点配置。
在_config.yml文件的最后，有Deployment字段，修改成

deploy:
  type: git
  repo: [your github page repository].git
  branch: master

注意，在配置文件中的:之后需要加一个空格处理。

部署Hexo

进如hexo文件夹，在终端输入
hexo g
生成页面
hexo d
部署页面到指定的服务器，这里就是我们的github page。快去http://evrins.github.io/看看你部署的hexo吧^.^
